# vk_bot_petka
API vk + yandexGPT

Данный бот был создан в качестве эксперимента с помощью Deepseek. Эксперимент заключается во внедрении боту мышления и личности со своими интересами для создания собеседника с ИИ на основе одной только истории переписок.
На данный момент (09.10.2025) бот поддерживает разговор в беседках.

## Новые возможности (Классификаторы YandexGPT)

С помощью классификаторов YandexGPT бот теперь может:
- **Анализировать намерения** сообщений (вопрос, утверждение, просьба, и т.д.)
- **Определять эмоциональный тон** (позитивный, негативный, нейтральный, шутливый)
- **Выбирать тип ответа** (фактический, творческий, поддерживающий)
- **Генерировать "мысли"** перед ответом для более осмысленного общения

Классификатор использует YandexGPT API для анализа сообщений и помогает боту лучше понимать контекст беседы.

Чтобы бот работал - необходимо:
1) Зарегистрировать его в ВК (если не хотите чтобы он писал от вашего имени)
2) Получить постоянный токен для АПИ ВК с помощью KateMobile - перейдите по ссылке и авторизуйтесь
https://oauth.vk.com/authorize?client_id=2685278&scope=1073737727&redirect_uri=https://oauth.vk.com/blank.html&display=page&response_type=token&revoke=1
3) Получить ID папки https://console.yandex.cloud/
4) Положить более 200руб на ваш платёжный аккаунт
5) Получить YANDEX_OAUTH_TOKEN
6) Вставьте всю необходимую информацию в файл .env

## Настройка классификатора

В файле `.env` вы можете настроить параметры классификатора:

```
# Включить/выключить генерацию мыслей (true/false)
USE_THOUGHT_GENERATION=true

# Показывать мысли в ответах бота (true/false)
# false - мысли используются внутренне для улучшения ответов
# true - мысли видны в ответах бота (режим отладки)
VERBOSE_THOUGHTS=false
```
